What is GPA?

GPA (Grade Point Average) is a standardized way to measure academic performance. This tool uses a common 4.0 scale (A = 4.0).

How it works

  1. Multiply each course's grade-point value by its credit value.
  2. Sum all grade points.
  3. Divide by the total credits attempted.
